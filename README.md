# Speccy OS E5 Ultra - Imperial Edition 🚀

**Speccy OS E5 Ultra** es un frontend de emulación de alto rendimiento diseñado para dispositivos Android, centrado en la optimización extrema del hardware y una experiencia de usuario inmersiva tipo consola.

Esta "Imperial Edition" (V 0.2.1b) introduce el **IA Architect Engine**, un sistema de optimización neural que adapta el comportamiento del kernel y los gobernadores de CPU/GPU en tiempo real según el sistema emulado.

---

## 🛠️ Características Principales

### 1. IA Architect & Hardware Control 🧠
- **Optimización Neural**: El motor Architect analiza la carga de trabajo y ajusta las frecuencias para mantener FPS estables con el mínimo calor.
- **Perfiles de Rendimiento**: 
  - `ECO`: Ahorro máximo para sistemas de 8/16 bits.
  - `BALANCED`: Equilibrio inteligente para emulación media (PSP, Dreamcast).
  - `PERFORMANCE`: Potencia sostenida para PS2, GameCube y Switch.
  - `EXTREME`: Modo Ultra desbloqueado para usuarios Pro (Xanite / Windows).
- **Telemetría en Tiempo Real**: Monitorización de temperatura del SoC, uso de RAM, carga de CPU y velocidad de ventiladores.
- **Control de Ventilación**: Gestión manual y automática de refrigeración activa para dispositivos compatibles.

### 2. Interfaz Inmersiva 🎮
- **Temas Dinámicos**: Cambia entre el modo `CLÁSICO` (tipo libro) e `INMERSIVO` (pantalla completa con fanart).
- **ADN Visual**: La interfaz adapta sus colores neón automáticamente según la plataforma seleccionada.
- **Dashboard Neural**: Acceso rápido a aplicaciones Android, ajustes del sistema y el panel de telemetría Imperial.

### 3. Biblioteca y Ecosistema 📚
- **Smart Sync**: Escaneo inteligente de ROMs con soporte para múltiples directorios y almacenamiento externo.
- **Scraper Integrado**: Obtención de carátulas, capturas y metadatos vía ScreenScraper y TheGamesDB.
- **RetroAchievements**: Integración total con logros para disfrutar de la vertiente competitiva de los clásicos.
- **Mapeado Universal**: Configuración de mandos físicos y táctiles con detección automática.
