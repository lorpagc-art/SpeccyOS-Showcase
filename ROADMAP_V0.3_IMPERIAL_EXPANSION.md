# Speccy OS V0.3 - La Expansión Imperial 🚩

Este documento detalla la visión estratégica para convertir Speccy OS en el Meta-Sistema Operativo universal para consolas portátiles y dispositivos móviles.

## 1. Quick-Access Overlay (Menú Radial In-Game) ⚡
*   **Concepto:** Menú lateral/radial accesible durante el juego mediante atajos de hardware (ej. `M + L1`).
*   **Funciones:**
    *   Ajuste en tiempo real de Brillo y Volumen.
    *   Cambio de perfiles térmicos (Silent/Balanced/Turbo).
    *   Monitor de FPS y Temperatura flotante.
    *   Botón de "Kill Task" para salida rápida de emuladores.

## 2. Zero-Setup Configurator (Auto-Inyección) 🛠️
*   **Concepto:** Configuración inteligente automática basada en el hardware detectado.
*   **Funciones:**
    *   Inyección de archivos `.ini` y `.cfg` optimizados para RetroArch, AetherSX2, PPSSPP, etc.
    *   Selección automática de la mejor resolución de renderizado según la GPU del dispositivo.
    *   Mapeado automático de controles según la base de datos de hardware.

## 3. Speccy Cloud Sync (Continuidad Universal) ☁️
*   **Concepto:** Sincronización de partidas entre dispositivos (Handheld <-> Mobile).
*   **Funciones:**
    *   Sincronización automática de `Saves` y `States` vía Google Drive / Dropbox.
    *   Sistema de "Continuar jugando" en el Dashboard que detecta el último estado en la nube.

## 4. Proactive Architect AI (IA Contextual) 🧠
*   **Concepto:** La IA deja de ser reactiva para ser proactiva.
*   **Funciones:**
    *   Detección automática de caídas de frames con sugerencias de optimización.
    *   Aviso de temperatura crítica con reducción automática de ciclos si es necesario.
    *   Recomendaciones de juegos basadas en la potencia del dispositivo y gustos del usuario.
